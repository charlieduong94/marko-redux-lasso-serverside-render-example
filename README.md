# marko-redux-lasso-serverside-render-example

This is a simple example of what serverside rendering with
[marko-redux](https://github.com/charlieduong94/marko-redux)
looks like.

## Installation

```sh
npm install

# or
yarn install
```

## Starting the app

```sh
npm start

# or
yarn start
```

Note: Go to `localhost:8080` to see the app.
